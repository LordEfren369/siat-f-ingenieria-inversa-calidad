# Ingeniería Inversa y Aseguramiento de Calidad del Sistema SIAT-F

## Descripción del proyecto

Este repositorio contiene la documentación, evidencias, diagramas UML y actividades correspondientes al proyecto de ingeniería inversa y aseguramiento de calidad de un software existente.

El sistema analizado es SIAT-F, Sistema de Información y Análisis para la Toma de Decisiones - Fuentes, una plataforma diseñada para apoyar la administración operativa y la toma de decisiones de la empresa Distribuidora Fuentes.

## Objetivo general

Aplicar ingeniería inversa al sistema SIAT-F para identificar su estructura, módulos principales, clases, relaciones y funcionamiento general, generando documentación técnica y evidencias que permitan apoyar el aseguramiento de calidad del software.

## Objetivos específicos

- Analizar la documentación y diagramas existentes del sistema SIAT-F.
- Identificar los módulos principales del sistema.
- Reconocer las clases, atributos, métodos y relaciones principales.
- Generar y organizar diagramas UML en formato PNG o JPG.
- Documentar hallazgos derivados del proceso de ingeniería inversa.
- Organizar las actividades mediante un tablero ágil Kanban.
- Preparar evidencias para la entrega del proyecto.

## Sistema analizado

SIAT-F es una plataforma de información y análisis para la toma de decisiones, diseñada para Distribuidora Fuentes, empresa dedicada a la distribución de productos en tiendas de Coatepec y sus alrededores.

El sistema tiene como finalidad transformar datos transaccionales cotidianos en inteligencia de negocios, apoyando la administración operativa diaria y la evaluación de escenarios para la toma de decisiones.

## Áreas principales del sistema

### Gestión operativa

El sistema permite administrar información comercial básica, incluyendo:

- Registro y control de clientes.
- Manejo de inventario.
- Control de productos.
- Seguimiento de pedidos.
- Registro de ventas.

### Inteligencia y análisis

El sistema integra herramientas para apoyar el análisis estratégico, tales como:

- Dashboard ejecutivo.
- Indicadores clave de rendimiento.
- Correlación de Pearson.
- Matrices de ponderación.
- Simulación de Monte Carlo.
