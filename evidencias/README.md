# Evidencias

Esta carpeta contiene evidencias del desarrollo, organización y seguimiento del proyecto.

## Evidencias sugeridas

- Captura del repositorio en GitHub.
- Captura del tablero Kanban en GitHub Projects.
- Captura del apartado UML.
- Capturas de los diagramas UML.
- Evidencias de avances y cambios realizados en el repositorio.

## Objetivo

Registrar visualmente el avance del proyecto y respaldar las actividades realizadas durante el proceso de ingeniería inversa y aseguramiento de calidad.
