# Apartado UML

En esta carpeta se almacenan los diagramas UML generados durante el proceso de ingeniería inversa del sistema SIAT-F.
https://github.com/LordEfren369/siat-f-ingenieria-inversa-calidad

## Diagramas incluidos

- Diagrama de clases general.
- Vista de administrador.
- Vista de vendedor.
- Modelo JavaReverse.
- Diagramas derivados del análisis del sistema existente.

## Formato de entrega

Los diagramas se entregan en formato PNG o JPG, de acuerdo con los requisitos establecidos para el proyecto.

## Objetivo

Representar visualmente la estructura del sistema SIAT-F, sus clases principales, atributos, métodos y relaciones.
